## ✨ Key Features
* **Interactive Weave Animation:** A custom CSS/JS loader that mimics the warp and weft of a loom.
* **Scroll-Triggered Reveals:** Sections elegantly slide into view as the user scrolls, utilizing the JavaScript Intersection Observer API.
* **Dye Diffusion Effects:** Background animations that simulate the chemical processing and dyeing of fabrics.
* **Fully Responsive:** Optimized for both desktop and mobile viewing.

## 🛠️ Built With
* **HTML5:** Semantic structure and accessible design.
* **CSS3:** Custom animations, Flexbox/Grid layouts, and root variables for consistent branding.
* **Vanilla JavaScript:** Lightweight scroll detection and DOM manipulation without heavy frameworks.
